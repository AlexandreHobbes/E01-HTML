Manual de Programação da linguagem C


Introdução


A linguagem C é uma linguagem estruturada criada em 1972 por Dennis Ritchie para ser usada no desenvolvimento sistema operacional Unix no Bell Labs, que na época pertencia à AT&T (hoje parte da Alcatel-Lucent). O "C" foi derivado da linguagem B, desenvolvida por Ken Thompson, que por sua vez se baseou-se na linguagem BCPL. Ele Foi desenvolvida por programadores para programadores tendo como meta características de flexibilidade e portabilidade. O "C" é uma linguagem que nasceu juntamente com o advento da teoria de linguagem estruturada e do computador pessoal. Assim tornou-se rapidamente uma linguagem “popular” entre os programadores. Ele foi usado para desenvolver o sistema operacional UNIX, e hoje esta sendo usada para desenvolver novas linguagens, entre elas a linguagem C++ e Java. Variáveis e expressões matemáticas


Em C, utilizamos as variáveis dos seguintes tipos:

  • Int: armazena números inteiros de valores compreendidos entre -32767 a 32767

  • Float: armazena números reais compreendidos entre -3.4 x1038 e 3.4 x 1038 com até 6 dígitos

  • Double: armazena números reais compreendidos entre -1.7 x 10308 até +1.7 x 10308 com até 10 dígitos

  • Char: armazena letras e símbolos

Em C, nós podemos executar operações aritméticas usando variáveis e constantes, sendo muito fácil utililá-las. As operações mais comuns são:
  • + Adição

  • - Subtração

  • * Multiplicação

  • / Divisão

  • % resto(módulo)


Podemos utilizar as variáveis e as operações de diferentes formas, conforme o código abaixo:
int a = 15;
int b = 72;
int c = a * b;
/* c valerá 15×72 */
